qrcode.js — vendored third-party library
========================================

Project:  qrcodejs (davidshimjs/qrcodejs)
Source:   https://github.com/davidshimjs/qrcodejs
Fetched:  https://raw.githubusercontent.com/davidshimjs/qrcodejs/master/qrcode.js
SHA-256:  3ee72de9f69c668f9567363a9358df955960bae9000d9ebd66414670f88e8735
License:  MIT

This is the library's original, unmodified, human-readable source — the
minified build is deliberately NOT shipped, so every line of JavaScript in this
plugin can be read as-is.

Used to render the 2FA enrollment QR code entirely client-side from the
otpauth:// provisioning URI, so the TOTP secret never leaves the browser and
no external QR service is contacted. Do not edit this file locally; re-vendor
from the source above if an update is needed and update the SHA-256 here.
